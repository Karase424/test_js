Облачное тестирование - это тип тестирования ПО, 
используя облачную инфраструктуру, сервесы через интернет,
а не локальные сервисы. Тут происходят проверки функциональности,
производительности и безопасности приложения.

Оно нужно для, например, проверки нагрузки на сайт;
как работает приложение на разных браузерах и ОС, 
отчего проверка происходит облачно и тебе не надо иметь их физически;
также есть ли задержки, нормально ли он локализируется и так далее.

Применять его можно для нагрузочного тестирования, например,
для сайтов-магазинов. Также может помочь для проверки адаптации,
то есть поплывёт ли вёрстка после изменения кода на разных устройствах.

Плюсы:
1) Экономия денег. Не нужно тратиться на покупку серверов и устройств.
2) Масштабируемость, то есть можно развернуть среду для тестирования любых размеров.
3) Большая доступность,так как тебе не надо иметь всё физически, а облачно
4) Можно сразу быстро всё проверить на нескольких устройствах.

Минусы:
1) Наверное безопасность, потому что идёт передача кода на сторонние сайты.
2) Зависит от интернета
3) Проблемы с конфигурацией, отчего в облаке может быть по другому, чем на самом деле.
